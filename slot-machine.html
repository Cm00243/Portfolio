Slot Machine

This slot machine program allows users to deposit money, place bets on multiple lines, and spin the reels to win or lose based on the outcome. Below is a GIF demonstrating the app in action.

Overview

The Slot Machine project enhanced my understanding of Python programming, including handling user input, implementing game logic, and simulating a slot machine's random outcomes.

Technologies Used

This project was a valuable learning experience, allowing me to apply theoretical knowledge in a practical setting. It improved my understanding of Python programming and how to create interactive terminal applications.
